# CLI objective preview
> **Status**: superseded-by: docs/Phase3/04_objective_preview_cli.md


When autonomy is disabled, the operator CLI has no dry-run or preview command for the metric-to-sub-goal proposal that autonomy would generate; `amaze objective disable` works and `amaze objective create` can create an active objective, but the closest available command mutates state instead of showing a non-mutating proposal preview.

See: docs/Phase3/04_objective_preview_cli.md
